### **Key Points for Revision**

- **Project Setup:**
    - **npm init**: Initializes a Node.js project.
    - **Express**: A lightweight framework to handle HTTP requests and responses.
    - **Nodemon**: Automatically restarts the server on file changes.
- **MongoDB Connection:**
    - Use **Mongoose** to connect to **MongoDB Atlas**.
    - Ensure proper IP whitelisting and user roles for secure access to MongoDB.
- **API Creation:**
    - **POST** route to create data and **GET** route to read data from MongoDB.
    - Use **Postman** to test API endpoints and verify if data is created and fetched correctly.
- **JSON Format:**
    - Data is sent and received in **JSON** format, which is lightweight and easily parsed by JavaScript.
